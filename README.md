# codeshare_sf
FastAPI project for code snippets storing and sharing


## Требования

```commandline
1. Приложение написано на языке Python с использованием фреймворка FastAPI и ORM SQLAlchemy.

2. Архитектура приложения выстроена в соответствии с принципами «чистой архитектуры». Проект состоит из файла main.py и директорий: core, api, schemas, models, services, db, tests.

3. В приложении реализована система аутентификации и авторизации пользователей по примеру, разобранному в уроке 4.

4. Авторизованный пользователь может создавать, смотреть, редактировать и удалять код-сниппеты.

5. Создав код-сниппет, пользователь может отправить ссылку или uuid другому пользователю. Другой пользователь может запросить код-сниппет по uuid без авторизации.

6. Все данные пользователей и код-сниппеты хранятся в базе данных. В приложении реализовано асинхронное подключение к базе данных по примеру, разобранному в уроке 3.

7. В приложении реализована система логирования и обработки ошибок по примеру, разобранному в уроке 5.
```
